## Description

Build the Redemption Code modal ‚Äî shown after a successful points redemption (triggered from the Redeem Points bottom sheet). Displays an animated checkmark, the redemption code in a styled code block (large monospace, accent border), the RM discount value, usage instructions, and a Done button that dismisses the modal and refreshes the points balance. Built as a variant of the `AppDialog` component from Phase 0.
